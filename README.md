## Install

### NPM:

```sh
npm install @compactjs/parse-time
```

### CDN:

```html
<script src="https://unpkg.com/@compactjs/parse-time/dist/index.umd.js"></script>
```

## Usage

### As module:

```javascript
import { parse } from '@compactjs/parse-time';
```

### Example:

```javascript
parse('13:30'); // => 13.5
```

## Run tests

```sh
npm run test
```
